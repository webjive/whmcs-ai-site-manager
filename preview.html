How to use AI Site Manager

Welcome to AI Site Manager! Use the chat to make changes to your website in plain English — no technical knowledge needed.

How it works:

  1. Type what you'd like to change in the chat box (e.g. "Change the headline to Welcome").
  2. Your AI assistant applies changes to a private staging area. The preview updates automatically.
  3. When happy, click ✅ Commit Changes to publish to your live website.
  4. Changed your mind? Click 🗑 Discard to cancel with no harm done.

Tips:

  • Attach images or files with the 📎 button.
  • Press Enter to send · Shift+Enter for a new line.
  • Use Fullscreen (top-right of the preview) for a larger workspace.
  • Use 🗑 Clear Chat to wipe history and start fresh.
